# AntiLag

**AntiLag** — оптимизатор задержек (latency / input lag) и сетевой утилиты для Windows 10/11.
Снижает задержки, держит таймер системы 0.5 ms, диагностирует железо и сеть,
и откатывает любые изменения в один клик.

> Автор: **AxelNerv** · Создано с Claude Code · Версия **0.4.2**

---

## ✅ Безопасно для анти-читов

AntiLag работает **строго в user-mode**: только официальные API Windows (таймер,
реестр, схемы питания). **Нет** драйверов ядра, инъекций, хуков и оверлеев —
поэтому он не конфликтует с Vanguard / VAC / EAC / Ricochet.

## Возможности

- **Базовое** — универсальные безопасные твики (таймер 0.5 ms, схема питания, отзывчивость системы), включены по умолчанию.
- **Игровые** — опциональные оптимизации (Nagle, приоритет переднего окна и т.д.), по умолчанию выключены.
- **Сеть**
  - *Состояние сети* — мониторинг соединения (задержка / потери / джиттер) + **тест скорости** в несколько потоков (RU-сервер Selectel или Cloudflare).
  - *Пинг сети* — Ping / TCPing / трассировка + диагностика ошибок сети.
- **DNS** — подбор и применение самого быстрого DNS-сервера.
- **DPC-метр** — приближённый замер задержки DPC (user-mode).
- **Ускорение** — очистка оперативной памяти.
- **Диагностика** — авто-проверка системы на проблемы с задержками + быстрый переход к нужному твику.
- **Драйверы** — проверка драйверов и поиск обновлений.
- **О системе** — бенто-дашборд железа + живые метрики (CPU / память / процессы / сеть).
- **Трей-режим** — держит 0.5 ms в фоне; автозапуск с Windows.

## 📥 Установка

1. Скачай [`release/AntiLag.exe`](release/AntiLag.exe).
2. Запусти **от имени администратора** (нужно для изменения системных настроек).

Это один self-contained файл — **.NET ставить не нужно**, всё внутри.
Работает на Windows 10/11 x64.

### «Система Windows защитила ваш компьютер» (SmartScreen)

При первом запуске Windows может показать синее окно SmartScreen с надписью
**«Неизвестный издатель»**. Это **нормально** для любой новой программы без платной
цифровой подписи — не признак вируса.

Нажми **«Подробнее» → «Выполнить в любом случае»**.

Если окна «Подробнее» нет: ПКМ по `AntiLag.exe` → **Свойства** → поставь галочку
**«Разблокировать»** внизу → **ОК**, затем запусти.

## 🛠 Сборка из исходников

Нужен [.NET 8 SDK](https://dotnet.microsoft.com/download).

```powershell
# обычная сборка
dotnet build src/AntiLagPro.App -c Release

# собрать один self-contained .exe (как в release/)
dotnet publish src/AntiLagPro.App -c Release -r win-x64 --self-contained true `
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true `
  -p:EnableCompressionInSingleFile=true -p:DebugType=none
```

> ⚠️ Все `.cs` / `.csproj` / `.xaml` сохранены в **UTF-8 с BOM** — иначе
> PowerShell 5.1 и Roslyn ломают кириллицу. После правок перекодируй обратно в UTF-8 BOM.
